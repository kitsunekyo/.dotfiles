managed with `GNU stow`

## Install

`brew install stow`

## Setup

Clone repo to home directory, to ensure the directory is `~/.dotfiles`.

Run `stow .` within `~/.dotfiles` to symlink all files.

## Packages

Ensure these are installed.

- "fzf"
- "nvim"
- "rg" (ripgrep)
- "fd" (fd-find)

For linux, install linuxbrew.
